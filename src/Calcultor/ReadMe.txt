-----------------All Classes of the project and their description------------------------

Cal : is a class that contains design and all needed attributes, and also configure events on each button.

Calc : to clear labels of the first number, operation and second number, also to set the chosen operation.

Numb : To eliminate using the dot twice and show the the typed numbers and operation on the graphic interface.

Result : do calculations and show the result.

Exe : Run all the code.
